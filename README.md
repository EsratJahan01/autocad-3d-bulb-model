# 3D Bulb with Filament (AutoCAD)

## Project Overview
This project demonstrates the creation of a **3D bulb model with filament using AutoCAD** as part of the **CSE200: Computer Aided Engineering Drawing** course.

The model was created using geometric modeling techniques and AutoCAD commands to construct the bulb glass, screw base, and filament structure.

## Project Components
The bulb model consists of three main parts:

1. Bulb Glass
2. Screw Base
3. Filament

Each component was designed separately and then assembled to create the final model.

## Geometric Dimensions

| Component | Dimension |
|----------|-----------|
| Bulb Height | 127 mm |
| Maximum Diameter | 50 mm |
| Screw Base Height | 32 mm |
| Filament Length | 96 mm |

*(Dimensions are approximate and can be adjusted.)*

## Modeling Process

### 1. Bulb Glass
- Created a 2D profile using Line and Circle commands
- Used the **Revolve** command to generate the 3D bulb shape

### 2. Screw Base
- Created the base using Polyline
- Used **Helix and Sweep** commands to generate threads

### 3. Filament
- Designed a spiral structure
- Converted the spiral path into a 3D solid using the **Sweep** command

### 4. Final Assembly
- Positioned components using **Move**
- Combined parts using the **Union** command
